# story-share-server
## 説明
- StoryShareのバックエンドサーバーです。
## 起動方法
- `python -m venv .venv`
- `source .venv/Scripts/activate`
- `pip install -r requirements.txt`
- `python manage.py runserver`
## 仕様技術
- フレームワーク
  - django（django-rest-framework）
- エディタ
  - Visual Studio Code
- lint等
  - Black Formatter
  - Flake8
  - isort
